!ИНСТРУКЦИЯ!
opencv.py - !ВАЖНО! основыной код, который запускает видео с веб-камеры, получает фото, обрабатывает его и перемещает в классификатор!
!!!ФАЙЛЫ И МОДЕЛИ!!!
1. ОСНОВНАЯ модель для нейронной сети по распознаванию и классификации лица - "main_model_for_opencv.h5"!!!
2. Файл "Имена", в нем содержатся все имена людей, на которых обучалась модель
3. Файлы "opencv_face_detector_uint8" и "opencv_face_detector"- это оновные модели! Именно на них настоящая модель проходила дообучение
4. captured_face.jpg - это фото, которое нейросеть получает с вебкамеры и отдает нейросети для классификации
!!!ПАПКИ!!!
1. dataset/train - в этой папке находятся все датасеты(уже обработанные фото для дообучения модели)
2. dataset/test - это данные для тестирования модели 
3. dataset_first - папка с рандомными фото разных людей(использовалось для тестирования)
!!!АКТИВНЫЕ ПРОГРАММЫ!!!
1. face.py - код, который получает из файла не обработанные фото, оставляет по центру лицо, переводит его в ЧБ, а так же уменьшает масштаб до 224х224 пикселя
2. learn.py - код, который дообучает модель на датасетах из папки dataset/train
3. main.py - код, который использовался для обучения тестовой модели
4. opencv.py - !ВАЖНО! основыной код, который запускает видео с веб-камеры, получает фото, обрабатывает его и перемещает в классификатор!
5. run.py - код, который позволяет без использования веб-камеры, проверить работоспособность модели
